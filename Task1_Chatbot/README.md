# Rule-Based Chatbot

## Description

This is a simple Rule-Based Chatbot developed using Python. It responds to user inputs based on predefined rules and keywords.

## Features

- Greets the user
- Answers simple questions
- Responds to greetings
- Gives predefined replies
- Exits when the user types `bye`

## Technologies Used

- Python

## Files

```
Task1_Chatbot/
│── chatbot.py
│── responses.py
│── README.md
```

## How to Run

1. Open the project folder.
2. Open Terminal.
3. Run the following command:

```bash
python chatbot.py
```
